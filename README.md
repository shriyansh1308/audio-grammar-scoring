# Audio Grammar Scoring System

This project evaluates the grammatical correctness of spoken English audio.

## Methodology

1. Speech-to-text conversion using Whisper
2. Grammar evaluation using LanguageTool
3. Score assignment based on detected grammar errors

## Files

- shlproject.ipynb → Main implementation
- train_results.csv → Training results
- test_predictions.csv → Predicted scores

## Description

The system processes audio files, converts them into text, analyzes grammar quality, and assigns a score. This demonstrates integration of speech recognition and natural language processing techniques.

Dataset provided as part of SHL assessment.
